Course Info

Description

In this four-week workshop, students learn about the structure of code to build websites using HTML/CSS and Javascript/jQuery. In-class demos and take-home projects gradually build up programming skills, while also teaching students how to reference code documentation to solve design challenges. Class lectures offer a broad overview of the design+technology field, including topics in computer science, digital design languages (e.g. Material Design), and related technologies like web frameworks (e.g. Bootstrap, React) and commercial platforms (e.g. Webflow, WordPress). Students leave the course as both better programmers and designers, with a new understanding of code’s ability to shape design decisions.
