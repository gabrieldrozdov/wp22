CSS is how you get things from looking *default* to looking *snazzy*.

  • You can add inline CSS to any element.
  • Look at the source code to see the notation needed. This notation is always in the starting tag for an element.
  • There is no limit to how many styles you apply to an element. If you want multiple styles, make sure to separate them by a semicolon.
  • The downsides of inline CSS are that you can’t reuse styles, and that things get very unorganized.
  • What if you want every list item to look exactly the same? That’s what we’ll cover in the next demo.
