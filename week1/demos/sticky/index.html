Let’s say that we want an element to follow the screen, but we don’t want it to overlap other elements. Fixed and absolute positioning wouldn’t work, because the elements would overlap. Here comes sticky positioning to the rescue!

This paragraph is inside a sticky Div. That means it’ll follow the screen as you scroll.

Sticky elements need to be inside of Divs (or other containers). The parent’s position needs to be set to "relative", and the sticky element needs to have a top, left, right, or bottom value to determine where it sticks to.
