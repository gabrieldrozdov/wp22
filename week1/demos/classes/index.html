Classes are the way you can apply the same styles to multiple elements.

This way, you don’t need to repeat code, and you can easily update the styling for multiple elements at once. It’s great!

This is what the normal paragraph element would look like without a class applied to it. Notice that you can also apply CSS to everything that uses a particular tag, like the "p" element. That’s why this text is monospaced right now.

Classes can be applied to anything. For elements like Divs, anything within the element will inherit the CSS styles of the parent’s class.

You can combine classes with inline CSS.

You can even combine multiple classes. Just separate them by a space!
