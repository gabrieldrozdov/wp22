There is a limited number of web-safe fonts that every computer will have, but that’s super boring. How can you load your own fonts?

If you have the font files, that’s great! You can include them in your repository and use the @font-face code to load in your own files. A whole bunch of file types are supported, but .woff and .woff2 are best for the web (even though I’m using .ttf fonts here). And here’s your custom font!

Another option is to load in fonts from an online source, like Google Fonts. This is also super easy!

After picking your font, Google will provide you with the code you need to include on your webpage to load in their fonts. I’ve added some lines to the "head" tag that do just this for the typeface Josefin Sans. Then, in your stylesheet you can specify the font you want to use and voila!
