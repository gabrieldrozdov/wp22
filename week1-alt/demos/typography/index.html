Typographic Hierachy

HTML is all about organizing information.

Different heading tags let you specify a hierarchy of headings, subheadings, and sections. You can also organize all of this information into Divs, Sections, or other organizing element tags.

There is no singular right way to use all of these tags

You should generally not skip heading levels (H1, H2, H3, etc.). The different levels are not about sizing, but about hierarchy.

HTML cannot do much on its own

If you’re getting frustrated with the limitations of HTML, good! It’s pretty useless without some CSS, which we’ll get into with the next demo.
