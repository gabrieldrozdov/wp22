We could have a Div inside another Div! The "text-align" property then lets us center things.

There are even better and easier ways to do this, but we’ll keep it simple for now.

What about the "position" property in CSS? By default, everything is pretty much set to "static" positioning. You’ve already gotten a sense about how that works.

Look at the bottom of the browser window to see "absolute" positioning.

Absolute positioning sets the position relative to some parent. In this case, the parent is the "body" tag.
This is a Div in a Div. The parent Div has position set to "relative", and the child has position set to "absolute". The child’s position will now be set relatively to the parent element.

What if you want something to stay still no matter what (including if the screen is scrolled)? That’s called "fixed" positioning! This page would look like a disaster with a fixed element on the screen, so you can test that on your own.
