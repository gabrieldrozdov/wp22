One exciting and new(ish) way to animate text is through variable fonts. Variable fonts are like the keyframed animations from the last demo, except that the keyframes are integrated into the font files themselves. For instance, this could mean that you can smoothly transition a font from a light weight to a heavy weight.

Where can you find variable fonts? The easiest place is Google Fonts!

Look at how cool this is!

(Hover over me!) You can also use hover selectors to affect variable fonts!

Different variable fonts have different variable sliders, so make sure your CSS code is appropriately targeting the right variable paramaters. For the font here (Raleway), there is only a "wght" parameter affecting font weight. You can see a whole range of other options here!
