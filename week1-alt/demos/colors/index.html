You can specify colors in a few different ways. Here are a few options!

  1. There are a bunch of colors you can specify by name directly. Here’s the list.
  2. Most commonly, you’ll use hex codes that start with a # symbol.
  3. Hex codes also support adding an extra two digits for opacity, but that gets a little confusing.
  4. RGB specifies the red, green, and blue values of the color you want. Websites don’t use CMYK because digital screens use RGB lights, not pigments.
  5. RGBA is the same as RGB but has an extra field for specifying the *opacity* of the background (from 0 to 1).
  6. HSL is hue, saturation, and lightness. Hue is 0 to 360, while saturation and lightness go from 0% to 100%.
