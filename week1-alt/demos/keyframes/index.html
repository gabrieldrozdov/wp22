With hover states, an element transitions from one set of CSS properties to another. What if we could do this automatically? We can!

Look at me go!

How did we do that? By using something called @keyframes! This lets us specify the CSS properties to interpolate between, so that an element can go from one set of properties to another automatically.

It works for any CSS properties!

Your animation can have multiple keyframes!

There are multiple CSS properties relating to keyframed animations, including ones that let you change the smoothing, the animation direction, the animation duration, and how many times the animations loops (if at all). Check them all out here!
